## [0.1.0a1] - 2024-12-22

### Added

- Initial alpha release
- Core reactivity system (Signal, Effect, Computed)
- UI components (Flex, Box, Text, Button, Input)
- Web renderer with WtfUIByte compiler
- TUI renderer with Flexbox layout engine
- CLI commands: `wtfui dev`, `wtfui build`, `wtfui new`, `wtfui learn`
- PyPI distribution as `pywtfui` (import as `wtfui`)
